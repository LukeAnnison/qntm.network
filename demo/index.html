qntm.network markdown demo

This page is rendered from markdown into HTML by a traced Python capability — site-renders-markdown, qntm.network's first package with a real call stack.

What it demonstrates

  • headings (three levels here)
  • lists, like this one
  • bold and italic
  • inline code
  • links

A fenced code block

from qntm_network.render import MarkdownRenderer

MarkdownRenderer().to_html("# hello")  # pure: markdown -> html
